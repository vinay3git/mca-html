HTML Lists

Ordered Lists :

Ordered lists are used when the order of items matters. They're created using the "ol" tag.

  1. First Item
  2. Second Item
  3. Third Item

Unordered Lists :

Unordered lists are used when the order of items is irrelevant. They're created using the "ul" tag.

  • First Item
  • Second Item
  • Third Item

Nested Lists

nested lists are lists fomed within lists:

  • Main Item 1
    • Sub item 1
    • Sub item 2
  • Main Item 2
